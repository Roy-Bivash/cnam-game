# FIP GAME

Card game with the CNAM univercity students as card

## Run
Run the backend first then the frontend

#### Backend

1. Navigate to the backend
```bash
cd backend
```
2. Install the dependency
```bash
npm install
```
2. Run the backend
```bash
npm run dev
```

#### Frontend

1. Navigate to the backend
```bash
cd frontend
```
2. Install the dependency
```bash
npm install
```
2. Run the backend
```bash
npm run dev
```


Open [http://localhost:5173](http://localhost:5173)
